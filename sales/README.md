# FutureFunded sales asset pack

This folder contains founder-ready assets for selling FutureFunded to youth teams, nonprofits, schools, clubs, and local sponsors.

## Files
- `founder-demo-script.md` — guided demo flow for live calls
- `outreach-email-pack.md` — cold outreach, warm outreach, follow-up, sponsor outreach
- `one-pager.md` — simple product story / summary sheet
- `objection-handling.md` — quick answers for common founder and sponsor objections
- `sponsor-meeting-script.md` — local-business pitch script for sponsor conversations
